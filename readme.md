# 📝 To-Do List App

## 📌 About
A simple To-Do List web app where users can add and remove tasks.

## 🚀 Features
- Add new tasks
- Delete tasks by clicking on them

## 🛠️ Technologies Used
- HTML
- JavaScript

## ▶️ How it Works
- User enters a task in the input field
- When "Add" button is clicked:
  - JavaScript creates a new list item (li)
  - Task is added to the list
- Clicking on a task removes it from the list

## 📂 How to Run
1. Open `index.html` in browser
